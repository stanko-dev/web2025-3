# Web2025-3: Lab Work #3 - Node.js Modules and Command-Line Tools

This repository contains the implementation of Lab Work #3 for the "Web Technologies" course (web2025). The project focuses on utilizing built-in and third-party Node.js modules, interacting with the file system, and processing JSON data from the National Bank of Ukraine (NBU) server.

## Purpose

The goal of this lab is to:

- Master the use of built-in Node.js modules.
- Learn to create and manage a `package.json` file for a project.
- Install and use third-party packages (e.g., Commander.js).
- Work with the command-line interface and file system in Node.js.

## Theoretical Background

- **Lecture 3**: Overview of selected JavaScript constructs and working with JSON in JavaScript.
- **Lecture 4**: Node.js modules and project structure, with examples of module usage.

## Project Structure

- `data.json`: JSON file containing data fetched from the NBU server (specific to the assigned variant).
- `package.json`: Project configuration file with dependencies.
- `index.js`: Main script implementing the required functionality.
- `output.txt`: Optional output file generated based on command-line arguments.

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended).
- [Git](https://git-scm.com/) for version control.
- Basic knowledge of JavaScript and command-line operations.
